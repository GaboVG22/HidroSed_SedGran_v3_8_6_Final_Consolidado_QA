# AUDITORÍA DE PRODUCCIÓN · HIDROSED SEDGRAN v3.7.2

## Correcciones aplicadas

- Se corrige el perfil 3D previo de secciones: ya no representa las secciones como líneas planas.
- El perfil 3D ahora dibuja la sección transversal real con `offset_m` y `z_m`.
- Se corrige la selección `Ver aceptadas`, `Ver rellenadas`, `Ver eliminadas`.
- Los puntos topográficos 3D se filtran según las secciones visibles.
- Se agrega ventana 2D en el módulo 5 Secciones para revisar cada sección en plano coordenado offset-cota.
- La ventana 2D muestra PK, km, ancho, cota mínima, cota máxima, profundidad geométrica y número de puntos.

## Diagnóstico del error visual

El gráfico previo estaba usando líneas verdes planas con z constante por sección. Eso servía como marca de ubicación, pero no como perfil transversal hidráulico. La nueva versión usa los puntos reales de cada sección (`section_points_df`) y dibuja el perfil transversal completo, permitiendo verificar si la geometría realmente representa un cauce con escurrimiento.
